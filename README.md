# 🚴‍♂️ Bike Sales Analysis - Data Analytics Project

## 🔍 Project Overview  
This project analyzes bike purchasing patterns using Excel. The dataset contains demographic and financial details of customers, and the goal is to uncover trends that influence bike purchases.

## 📊 Dataset Details  
The dataset includes:
- **Marital Status** (Married/Single)
- **Gender**
- **Income** (Annual income in USD)
- **Children** (Number of children)
- **Education** (Education level)
- **Occupation**
- **Home Ownership** (Yes/No)
- **Number of Cars Owned**
- **Commute Distance**
- **Region**
- **Age & Age Bracket**
- **Purchased Bike** (Yes/No)

## 🔄 Data Processing Steps  
1. **Data Cleaning**: Removed inconsistencies and formatted the dataset.  
2. **Pivot Table Analysis**: Summarized key insights based on gender, income, and bike purchases.  
3. **Dashboard Creation**: Visualized trends through charts and graphs.  

## 📈 Key Insights  
- Higher-income individuals are more likely to buy bikes.  
- Single individuals purchase bikes more frequently than married ones.  
- Commute distance influences the likelihood of buying a bike.  

## 📚 How to Use This Project  
1. Download the **Excel file** from this repository.  
2. Open the workbook to explore cleaned data, pivot tables, and dashboards.  
3. Modify the data or build upon the analysis for further insights.
